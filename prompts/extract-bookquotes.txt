You are extracting marked passages from a scanned book page. A separate
observation pass has already identified the hand-drawn marks on this page,
combining results from multiple views (full, left half, right half, rotated).
Your job is to use the combined observation report together with the page
images to produce well-formed entries.

YOU ARE GIVEN
- The CURRENT PAGE image
- The PREVIOUS PAGE image and/or NEXT PAGE image (when available) — for context
  only, used to complete a passage that overflows a page boundary
- A combined observation report listing every mark detected across the variant
  views of the CURRENT PAGE

WHAT TO DO
1. The observation report may contain duplicate or near-duplicate entries
   because each variant was observed independently. DEDUPLICATE: if two MARK
   blocks describe the same mark (same margin/position/shape, overlapping
   extent), treat them as one mark.
2. For each unique mark, locate the corresponding passage on the CURRENT PAGE
   image using the EXTENT hints.
3. Read the exact verbatim text of the passage. If the marked passage clearly
   continues onto the next page or began on the previous page, use those
   context images to read the full passage. The CENTRE OF GRAVITY of the
   passage determines the page assignment — but for this run, every entry
   you produce belongs to the CURRENT PAGE because the observation report
   is about the current page only.
4. Produce one output entry per unique mark.

If the observation report says the page has no marks (all variants reported
NO MARKS or the report is empty of MARK blocks), output exactly: NONE

OUTPUT FORMAT — one block per marked passage:

PAGE: <page label>
MARK TYPE: <restate the margin/position/shape from the observation>
PASSAGE: <the exact verbatim text the mark points to or underlines>
CONTEXT: <1-2 sentences of surrounding text so the passage stands on its own>
WHY NOTABLE: <brief interpretation of why a reader might have marked this>

Separate entries with a blank line.

Output ONLY entries in the format above (or NONE). No preamble, no commentary.

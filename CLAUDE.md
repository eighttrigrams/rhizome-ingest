If you want to write Python scripts, help yourself. Try to not install packages
but use ephemeral runs using `uv`.

Use all the available Rhizome Skills:
- rhizome-rest-api
- rhizome-books
- navigate-rhizome-user-interface (probably not needed)

Make sure you use the `rhizome-cli` script talked about in `rhizome-rest-api` instead of curl
to access Rhizome's REST API.

"Projects" for ingesting particular books are in the `data` folder (which is gitignored), under 
names corresponding to book titles.
